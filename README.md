## 使用

### 开发

只需要执行以下命令, 浏览器会自动打开

```bash
pnpm dev
```

### 构建

构建该应用只需要执行以下命令

```bash
pnpm build
```

然后你会看到用于发布的 `.output` 文件夹被生成。
